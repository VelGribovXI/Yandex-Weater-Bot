# Weather Bot 🌤️

Telegram бот для получения погоды с кэшированием.

## Технологии
- Python 3.11
- aiogram 3.x (асинхронность)
- aiohttp (запросы к API)
- OpenWeatherMap API

## Функции
- Погода в любом городе мира
- Кэширование на 10 минут
- Красивое форматирование
- Обработка ошибок

## Запуск
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать .env с токенами
4. Запустить: `python main.py`
